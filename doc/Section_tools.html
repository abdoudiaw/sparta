7. Additional tools

SPARTA is designed to be a computational kernel for performing DSMC computations. Additional pre- and post-processing steps are often necessary to setup and analyze a simulation. A few additional tools are provided with the SPARTA distribution in the tools directory and are described briefly below.

The file binary2txt.cpp converts one or more binary SPARTA dump files into ASCII text files. The syntax for running the tool is 

binary2txt file1 file2 ...

which creates file1.txt, file2.txt, etc. This tool should be compiled on a platform that can read the binary files created by a SPARTA run, since binary files are not always compatible across all platforms.
